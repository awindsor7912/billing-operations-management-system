# Billing Workflow

## Overview

The billing workflow is designed to convert report intake into a structured operational process.

Each report moves through defined workflow stages until billing is completed.

---

## Workflow Stages

New Intake  
Parsed  
Routed  
In Processing  
Awaiting Documentation  
Completed

---

## Operational Benefits

Structured workflow stages enable:

• clear task ownership  
• consistent report handling  
• operational transparency  
• improved compliance monitoring
